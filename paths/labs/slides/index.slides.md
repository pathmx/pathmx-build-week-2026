# An Introduction to PathMX

---

## What is PathMX?

- **Path** **M**arkdown e**X**tension
- A methodology for building curriculum with code
- A framework/build tool for creating interactive learning experiences

---

## Why use PathMX?

- Portable, local-first content
- Collaborative authoring using normal source control tools
- Deployable, self-contained learning experiences
- Works as an automatic agent harness
