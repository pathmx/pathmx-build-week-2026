# Cburnett Chess Pieces

These SVG pieces are vendored from the Lichess `public/piece/cburnett` set:

https://github.com/lichess-org/lila/tree/master/public/piece/cburnett

Lichess credits the set to Colin M.L. Burnett and lists
`public/piece/cburnett` as GPLv2+ in its `COPYING.md`:

https://github.com/lichess-org/lila/blob/master/COPYING.md

The files are used here as the standard/default chess piece set for the
reviewed literate chess lab. This lab includes only the original SVG files; it
does not redistribute generated atlas assets.
