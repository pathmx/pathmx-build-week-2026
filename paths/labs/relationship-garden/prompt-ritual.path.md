# Relationship Garden Prompt Ritual

These five prompts keep the garden local, intentional, and small. Replace the
angle-bracket placeholders before using a prompt. Codex should rely only on
written notes and should say when the files do not establish something.

---

<!-- id: begin -->

## 1. Begin

Use this for one calm weekly glance.

```text
Help me tend my Relationship Garden for the internship bed. Read only paths/labs/relationship-garden/people/ and garden.people.json. Suggest one human to contact and why, using written notes only.
```

---

<!-- id: capture -->

## 2. Capture

Use this after you deliberately decide a new relationship belongs in the
garden. Keep the rough note short and remove private details that do not help
future-you.

```text
I met someone. Draft a new people/<slug>.person.md and a matching garden.people.json entry from this note: """ … """
```

---

<!-- id: parallel-lives -->

## 3. Parallel lives

Use this when one human appears in multiple parts of life.

```text
Add or clarify parallel lives for <name> without flattening context. Warn me if a proposed ask mixes lives badly.
```

---

<!-- id: return -->

## 4. Return and persist a touch

Use this after a real interaction. It is the explicit bridge from temporary
Player state to durable Source.

```text
I tended <name> on <date> via <channel>: "<note>". Update lastTouched, Touches, freshness fields in both the person Source and JSON mirror. Propose a next move only if one is needed.
```

---

<!-- id: fade -->

## 5. Fade without guilt

Dormancy is a valid relationship state, not a failure screen.

```text
Help me gracefully mark <name> dormant — no guilt. Keep the origin story. Set a calm next review date if useful.
```

---

## Guardrails

- Never scrape an inbox, calendar, contacts list, or social network.
- Do not invent closeness, permission, promises, or personal details.
- Keep one primary next move, and allow “no task” to be the honest result.
- Update a person Source and its JSON mirror together.
- Prefer stewardship language: tend, return, rest, and reconnect.
