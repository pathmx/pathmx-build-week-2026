# Self-Learning Manual Test

## 1. Scaffold a new learning project

```bash

# create a new project folder
mkdir ~/<your-project-folder-path>
cd ~/<your-project-folder-path>

# make sure pathmx is up to date
pathmx self-update
pathmx init --template self-learning
pathmx play -- open
```

## 2. Use an agent to learn something new

### Codex Desktop

1. Create a new project in Codex and select the folder created above.
2. Create a new thread in the project enter `/path`. If all goes well, the agent should take it from there.
